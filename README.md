# cnweb-pg-guide

## 项目简介

本仓库旨在收集、归档并发布一系列独立的 HTML 页面。这些页面内容涵盖与中文网络环境相关的技术参考、工具索引及信息整理，以静态页面形式呈现，便于直接浏览或离线使用。

仓库不针对任何特定域名或网站，仅作为通用页面存档与展示用途。

## 目录结构

```
cnweb-pg-guide/
├── pages/           # 存放所有独立 HTML 页面
├── assets/          # 页面引用的静态资源（图片、样式表、脚本等）
├── index.html       # 仓库首页 / 页面导航
└── README.md        # 本说明文件
```

- `pages/`：每个子目录或文件对应一个独立页面，文件名或目录名反映页面主题。
- `assets/`：按资源类型分类，确保页面可独立运行。
- `index.html`：提供页面列表与快速访问链接。

## 页面归档说明

- 所有页面均为纯静态 HTML，无需后端支持。
- 页面内容遵循通用技术文档风格，不包含商业推广或诱导性信息。
- 页面可能引用外部资源（如 CDN 库），但尽量保持自包含或提供本地替代。
- 页面更新会通过 Git 提交记录体现，不另行发布版本号。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 建议新增页面或修正现有内容。
- 新增页面请放置在 `pages/` 目录下，并确保文件名简洁明了。
- 提交前请检查页面无外部依赖异常，并保持内容中立、客观。
- 本仓库为归档性质，不承诺持续更新频率，但会定期审核合并贡献。

## 使用方式

1. 直接访问本仓库的 GitHub Pages 站点（如已启用），或
2. 克隆仓库到本地，在浏览器中打开 `index.html` 或 `pages/` 下任意页面。
